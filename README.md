# MHGL

Це проста браузерна гра, у якій гравці керують персонажами та змагаються за перемогу. Гра використовує HTML, CSS та JavaScript, має модульну структуру коду та підтримує різні ігрові стани, моделі гравців і м'яча, а також інтерфейс користувача з таблицею лідерів та звуковими ефектами.

## Основні особливості
- Керування персонажами та м'ячем
- Відстеження стану гри
- Таблиця лідерів
- Звуковий супровід
- Зручний інтерфейс

## Запуск
Відкрийте файл `index.html` у браузері.
